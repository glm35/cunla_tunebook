According to http://lilypond.org/web/devel/participating/:

Submitting patches

If you want to contribute fixes of any kind, use the following procedure

    * Check out the latest sources via GIT at git.sv.gnu.org

          git clone git://git.sv.gnu.org/lilypond.git
          

    * Make your changes,
    * Create a patch:

            git commit -a 
            git-format-patch HEAD^
          

    * Send an email to lilypond-devel@gnu.org, including or attaching
    * the patch.
